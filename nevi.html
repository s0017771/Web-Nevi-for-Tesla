카카오 지도를 쓰려면 본인 키가 필요해요. 5분이면 됩니다. 한 번만 하면 끝입니다.

1. 카카오 개발자 앱 만들기

  1. developers.kakao.com 로그인 → 내 애플리케이션 → 애플리케이션 추가
  2. 만든 앱 → 앱 키에서 JavaScript 키와 REST API 키를 복사해 둡니다.

2. 도메인 등록 (이거 빠지면 지도가 안 떠요)

  1. 앱 → 플랫폼 → Web → 사이트 도메인에 이 주소를 추가:
  2. https://...vercel.app

3. 키 붙여넣기

  1. GitHub에서 nevi.html 편집 → 맨 위 CONFIG 부분의
  2. KAKAO_JS_KEY 와 KAKAO_REST_KEY 에 복사한 키를 넣고 커밋하세요.

REST 키는 도로 경로(길찾기)에만 쓰입니다. JS 키만 넣어도 지도+내 위치+목적지 검색은 동작해요.

키를 넣고 다시 열면 이 화면은 사라집니다.
